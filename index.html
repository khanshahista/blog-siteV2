About The Project

This project is a fully custom-built tech blog developed using HTML, CSS, and JavaScript.

It dynamically loads blog content from a JSON data file, with real-time search and animated modal popups for expanded reading.

Each blog post is visually enhanced with images fetched from the Pexels API, based on the keyword of the post.

Users can leave comments, which are stored in the browser using localStorage â€” keeping the experience interactive without a backend.

The interface includes scroll-triggered animations, smooth transitions, and a fully responsive layout that adapts to all devices.

Dark mode support is integrated to enhance visual comfort and offer a better reading experience, especially in low-light environments.

This project reflects my real-world frontend development skills, including clean UI/UX design, API integration, and performance-focused coding.

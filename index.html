About The Project

The OS Algorithm Simulator is a fully responsive and interactive web application designed to visually demonstrate fundamental operating system algorithms.

It includes simulations for both CPU scheduling algorithms like First-Come, First-Served (FCFS) and Shortest Job First (SJF), as well as memory and disk management algorithms such as FIFO, LRU, SSTF, SCAN, C-SCAN, LOOK, and C-LOOK.

Built entirely using HTML, CSS, and JavaScript, the project is frontend-only with no dependencies on external frameworks or backends, making it lightweight and fast-loading.

Each algorithm is presented inside an elegant card layout. Users can enter inputs like burst times or reference strings and instantly visualize the execution with results such as waiting times, page hits/misses, or seek sequences.

This project is intended for students, educators, and enthusiasts who want to understand OS concepts practically rather than just theoretically.

Features like a dark mode toggle, comment section, smooth scroll animations, and search functionality enhance the user experience and accessibility.

Ultimately, the project aims to bridge the gap between abstract textbook knowledge and hands-on learning, encouraging interactive and visual education in computer science.
